# ThynkTech-India-Data-analyst-projects
A collection of data analysis projects developed during my Data Analyst internship at Thynk Tech India. The work focuses on real-world business datasets involving data cleaning, data transformation, exploratory data analysis, and dashboard creation using Excel, SQL, Python, and Power BI ,Tablaueto support data-driven decision making.

This repository contains the projects and dashboards developed during my Data Analyst Internship at Thynk Tech India.
The internship focused on data cleaning, analysis, and visualization using modern analytics tools and databases to generate business insights.

During the internship, I worked on multiple real-world datasets and built interactive dashboards to analyze notifications, attendance, sales performance, marketing campaigns, and website user behavior.
